üß† Brainfuck Commands

> Move pointer to the right
< Move pointer to the left
+ Increment value at current cell
- Decrement value at current cell
. Output character at current cell
, Input character into current cell
[ Start loop (jump forward if value is 0)
] End loop (jump back if value is not 0)

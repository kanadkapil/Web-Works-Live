Windows Command Prompt Commands

Command Description
`dir` Lists the files and directories in the current directory.
`cd` Changes the current directory.
`cls` Clears the command prompt screen.
`exit` Exits the command prompt.
`echo` Displays a message or turns command echoing on/off.
`mkdir` Creates a new directory.
`rmdir` Removes an empty directory.
`del` Deletes a specified file.
`tasklist` Displays a list of all running processes.
`taskkill` Ends a running process by name or process ID.
`chkdsk` Checks the disk for errors and repairs them if possible.
`ipconfig` Displays the network configuration for your machine.

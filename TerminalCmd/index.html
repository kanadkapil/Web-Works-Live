Windows Command Prompt Commands

Command Description
`dir` Lists the files and folders in the current directory.
`cd` Changes the directory. Example: `cd C:\Users\Username\Documents`.
`cls` Clears the screen.
`exit` Closes the Command Prompt window.
`echo` Displays a message or turns command echoing on/off. Example: `echo Hello World`.
`pause` Pauses the command line processing and waits for a key press.
`mkdir` Creates a new directory (folder). Example: `mkdir NewFolder`.
`rmdir` Removes a directory. Example: `rmdir OldFolder`.
`del` Deletes a file. Example: `del file.txt`.
`copy` Copies files from one location to another. Example: `copy file.txt D:\Backup`.
`move` Moves files from one location to another. Example: `move file.txt D:\Backup`.
`ren` Renames a file or folder. Example: `ren oldname.txt newname.txt`.
`systeminfo` Displays detailed information about the system.
`hostname` Displays the computer's hostname.
`tasklist` Displays a list of all running processes.
`taskkill` Ends a running process. Example: `taskkill /PID 1234`.
`ipconfig` Displays IP configuration of the computer.
`ping` Tests the connection to a remote network host. Example: `ping google.com`.
`chkdsk` Checks the disk for errors.
`sfc /scannow` Scans and repairs system files.
`netstat` Displays active network connections.
`tracert` Traces the route to a network host. Example: `tracert google.com`.
`nslookup` Queries DNS servers for domain information.
`netsh` Configures network settings.
`ftp` Starts FTP client to transfer files.
